# TFTP Control Panel & Manager
Интерактивное Bash-меню для удобного управления сервером tftpd-hpa. Позволяет забыть о ручном вводе команд в терминале и предоставляет удобный интерфейс для настройки путей, мониторинга логов и управления файлами.

## 🚀 Основные возможности
* Управление сервисом: Старт, стоп и перезапуск одной кнопкой.
  
* Живой мониторинг: Просмотр трансфер-логов в реальном времени через journalctl.
  
* Управление путями: Быстрая смена рабочей директории сервера прямо из меню.
  
* Менеджер файлов: Встроенный запуск Midnight Commander (mc) с правами суперпользователя для управления прошивками.
  
* Автоматизация: При установке пакета сервер автоматически переводится в ручной режим (отключается автозапуск), чтобы не занимать ресурсы зря.
  
* Интеграция с Desktop: Создает ярлык в меню приложений с иконкой и запуском через pkexec.

## 📦 Сборка .deb пакета
Для создания установочного файла выполните следующие шаги:

Склонируйте репозиторий и подготовьте структуру:

```Bash
git clone https://github.com/withersky/tftpd-managerr.git
cd tftpd-manager
```

Соберите пакет:

```Bash
dpkg-deb --build ./tftpd-manager
```

Установите полученный пакет:

```Bash
sudo apt install ./tftpd-manager-*.deb
```

## 🛠 Зависимости
При установке пакета следующие зависимости будут установлены автоматически:

* `tftpd-hpa` — сам TFTP сервер.
  
* `mc` — консольный файловый менеджер.
  
* `systemd` — для управления службой.
  

## 📂 Структура проекта
* `usr/local/bin/tftpd-manager.sh` — основной управляющий скрипт.
  
* `usr/share/applications/tftpd-manager.desktop` — файл ярлыка для системного меню.
  
* `DEBIAN/postinst` — скрипт послеустановочной настройки (отключает автозапуск `tftpd-hpa`).

## 🖥 Использование
После установки найдите в меню приложений `TFTP Server Control`.

Заметка: При запуске система попросит пароль (через pkexec) для получения прав на управление системной службой и изменения конфигурационных файлов.
